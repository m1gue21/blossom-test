# Greycliff CF Font Files

Place the Greycliff CF font files here with these exact names:

- `GreycliffCF-Light.woff2`
- `GreycliffCF-Light.woff`
- `GreycliffCF-Regular.woff2`
- `GreycliffCF-Regular.woff`
- `GreycliffCF-Medium.woff2`
- `GreycliffCF-Medium.woff`
- `GreycliffCF-DemiBold.woff2`
- `GreycliffCF-DemiBold.woff`
- `GreycliffCF-Bold.woff2`
- `GreycliffCF-Bold.woff`
- `GreycliffCF-ExtraBold.woff2`
- `GreycliffCF-ExtraBold.woff`

You can purchase/download Greycliff CF at:
https://connary.com/greycliff.html

After adding the files, rebuild the Docker container:
  docker-compose build frontend && docker-compose up -d frontend
